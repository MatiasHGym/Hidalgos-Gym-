# Yoga Hidalgo´s GYM

Primera suite local para registrar clientes y asistencias del gimnasio.

## Incluye

- Rutas separadas por area: Yoga, Boxeo y Pesas.
- Panel administrador local en `admin.html`.
- Asistencias por clase de martes y jueves.
- Tres horarios por dia: 10:00-11:00, 11:15-12:15 y 12:30-13:30.
- Clientes con nombre, RUT, sexo, edad, complicaciones y uno o mas horarios.
- Resumen de clientes con ficha, modificacion y eliminacion.
- Datos guardados localmente en IndexedDB del navegador.

## Siguiente paso

Para tener un administrador accesible desde cualquier lugar y que todos los profesores escriban en el mismo lugar, hay que conectar esta base a una base de datos central, por ejemplo Supabase o Firebase. Esta version local sirve como maqueta funcional para probar flujo, pantallas y datos antes de subirla.
